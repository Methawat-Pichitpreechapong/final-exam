# Delivery Program

## Project Overview
This project is about the delivery system that user will have to write 2 customers names, and 1 driver name
then user have to write the items that the users want to order, and the system will show up all of the information that the users need to know.

## Features
- The status that shows the order(s) has been preparing or has been delivered
- Show Driver's name

## How to run
1.Click run
2.Write the name of customer
3.Write the item name that you want to order
